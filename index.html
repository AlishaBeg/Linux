# usermod

The usermod command in linux is used to modify a user account setting like username, group, membership, directory, shell and more.

change old username to new username

sudo usermod -l new_user_name old_user_name

set a new home directory for a user account

sudo usermod -d /home/username_directory username

set new login shell

sudo usermod -s /bin/shell user_name

lock user account

sudo usermod -L user_name

unlock user account

sudo usermod -U user_name

# change mode

chmod u+x file_location this command is enable user execute file permission & chmod 644 file_location this numerical command for user read + write group or other readonly file permission.

chmod [operation] file_location

type

  • (u) : user
  • (g) : group
  • (o) : other
  • (a) : all

permission

  • ( ) : none
  • (r) : read
  • (w) : write
  • (x) : execute

case

  • (+) : enable
  • (-) : disable
  • (=) : set

numeric

  • 0: none
  • 1: execute
  • 2: write
  • 4: read
